## 概要

林高勲(2019)「x86系コンピュータを動かす理論と実装 作って理解するOS」
に則ってOSを自作しています。

## 環境

- macOS Catalina
- QEMU

### QEMUのInstall方法

```sh
brew install qemu
```

## 準備

次のコマンドを実行します。

```sh
echo export PATH=\$PATH:~/prog/env/ >> ~/.zshenv
source ~/.zshenv
```
